# 인공지능을 이용한 자동 거래 등록 중고 도서 플랫폼
중고 도서 거래의 번거로움과 신뢰 문제를 해결하는 스마트 플랫폼

<br>

## 🎯 프로젝트 개요

중고 도서 거래를 할 때 판매자들이 가장 큰 불편을 느끼는 부분은 **책 정보를 일일이 입력해야 하는 번거로움**과, **도서 상태를 주관적으로 판단하면서 발생하는 분쟁**입니다.

특히 여러 권의 책을 한 번에 등록할 때 바코드를 하나씩 스캔하거나 직접 정보를 입력하는 과정이 너무 힘들어 판매를 포기하는 경우가 많았습니다. 또한 “상태 좋음”, “깨끗함” 같은 주관적인 표현 때문에 구매자와 판매자 사이에 신뢰 문제가 자주 발생했습니다.

**Rebook**은 이러한 문제를 해결하기 위해 만들어졌습니다.

**사진 한 장**만 찍으면 AI가 책 제목, 저자, ISBN 등의 정보를 자동으로 추출하고,  
**AI 이미지 분석**을 통해 도서의 외관 상태를 객관적으로 판정해줍니다.

복잡한 입력 과정 없이 빠르고 정확하게 중고 도서를 등록하고 거래할 수 있는 새로운 경험을 제공합니다.

<br>

## 📊 시장 분석

- 글로벌 도서 시장 규모: 약 **1,362억 달러**
- 국내 중고 도서 시장 규모: 약 **1조 원**
- 지역 기반 중고 거래 수요 급증 (지역 커뮤니티 서비스 이용률 25% 이상 증가)
- 중고 거래 이용자들의 주요 불만: **상품 상태 불일치** 문제
- 다권 판매 시 **바코드 개별 스캔**의 번거로움으로 인한 판매 포기 사례 다수

이러한 불편 요소들은 중고 도서 시장에서 여전히 해결되지 않은 핵심 문제입니다.

<br>

## 🔍 기존 서비스와의 차별점

| 구분              | 기업 매입 서비스           | 일반 C2C 플랫폼                | **Rebook** |
|-------------------|----------------------------|--------------------------------|------------|
| 거래 속도         | 빠름                       | 보통                           | 빠름       |
| 사용자 편의성     | 대면 검수 필요             | 개별 입력 필요                 | **최고**   |
| 상태 판단 방식    | 전문가 주관적 검수         | 판매자 주관적 판단             | **AI 객관적 판정** |
| 다권 등록         | 불편                       | 매우 불편 (바코드 개별 스캔)   | **사진 한 장으로 묶음 자동 등록** |
| 신뢰도            | 중간                       | 낮음                           | **높음**   |

### Rebook만의 핵심 차별점
- **AI 기반 객관적 도서 상태 판정**으로 구매자와 판매자 간 등급 분쟁 사전 방지
- **사진 한 장으로 여러 권 동시 등록** (묶음 자동 등록 기능)
- 주관적 표현이 아닌 **정량화된 AI 상태 평가** 제공
- 매끄러운 사용자 경험과 ‘지식의 순환’이라는 사회적 가치 실현

Rebook은 단순한 중고 거래 플랫폼이 아닌, **기술로 신뢰를 만들고 감성으로 지식을 연결하는** 차별화된 서비스입니다.
